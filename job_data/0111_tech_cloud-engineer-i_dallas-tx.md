# Cloud Engineer I

**Company:** Rippling | **Category:** Tech | **Location:** Dallas, TX | **Salary:** $74,000–$93,000 | **Type:** Full-Time | **Posted:** 2026-01-10

## About Rippling

Rippling is the first and only global HR platform that brings together HR, IT, and Finance on one unified software. We're building the modern operating system for how companies manage their people and technology.

## About the Role

As a Cloud Engineer I at Rippling, you'll be responsible for building and maintaining the cloud infrastructure that powers our platform. You'll work with a team of skilled engineers in a fast-paced, growth-focused environment. This remote role offers the opportunity to develop expertise in cloud architecture and DevOps practices.

## Responsibilities

- Design and implement cloud infrastructure on AWS or GCP, following best practices for security and scalability
- Manage containerization using Docker and Kubernetes; implement CI/CD pipelines
- Monitor system performance, troubleshoot issues, and implement optimizations
- Collaborate with development teams to support deployment and scaling requirements
- Maintain infrastructure documentation and implement infrastructure-as-code practices

## Requirements

**Education:** Bachelor's degree in Computer Science, Engineering, or equivalent experience

**Experience:** 1-3 years of cloud engineering or DevOps experience

**Skills:**
- Experience with AWS or GCP cloud platforms
- Docker and Kubernetes fundamentals
- Proficiency with Linux and bash scripting
- Understanding of networking, databases, and system architecture
- Familiarity with infrastructure-as-code tools (Terraform, CloudFormation)

**Certifications:** AWS Cloud Practitioner or Associate certification (preferred)

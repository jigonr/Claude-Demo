## About Gusto

Gusto is building modern payroll and HR software for small and medium businesses. Our platform serves hundreds of thousands of companies, managing payroll, benefits, and compliance for millions of employees.

## About the Role

We're hiring a Test Automation Engineer to design and implement automated test frameworks that ensure quality across our platform. You'll work closely with product and engineering teams to identify testable scenarios and build robust test suites that catch regressions before they reach customers.

## Responsibilities

- Design and implement automated test scripts using frameworks like Selenium or Cypress
- Develop test strategies to balance automation coverage with testing efficiency
- Collaborate with developers to identify critical user paths that need automated coverage
- Maintain and refactor test code as product features evolve
- Execute manual testing for exploratory and complex scenarios
- Participate in sprint planning and help estimate testing effort for new features

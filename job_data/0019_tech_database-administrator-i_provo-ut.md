**Company:** Cisco | **Category:** Tech | **Location:** Provo, UT | **Salary:** $55,000–$69,000 | **Type:** Full-Time | **Posted:** 2026-01-18

## About Cisco

Cisco is the worldwide technology leader in networking that transforms how people connect, communicate, and collaborate. Our infrastructure powers the digital backbone of organizations globally.

## About the Role

As a Database Administrator I at Cisco Provo, you'll manage databases that support our networking products and global infrastructure. You'll work in a hybrid arrangement with experienced database professionals, developing expertise in enterprise database administration at scale.

## Responsibilities

- Administer SQL Server and Oracle databases; perform patches, updates, and maintenance
- Monitor database performance and implement optimization strategies
- Execute and test backup and recovery procedures regularly
- Manage user access and database security policies
- Support development teams with schema design and performance tuning

## Requirements

**Education:** Bachelor's degree in Computer Science, IT, or related field; or equivalent experience

**Experience:** 1-3 years of database administration or systems administration

**Skills:**
- SQL Server or Oracle database administration
- Windows Server and Linux system administration
- T-SQL or PL/SQL query writing
- Backup and recovery technologies
- Strong troubleshooting and problem-solving abilities

**Certifications:** SQL Server or Oracle certification (preferred)

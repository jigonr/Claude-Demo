**Company:** Block | **Category:** Tech | **Location:** Provo, UT (Remote) | **Salary:** $63,000-$80,000/year | **Type:** Full-time | **Posted:** Mar 25, 2026

## About Block

Block is a financial services platform for creators and sellers. We empower people with financial opportunity—providing the tools, capital, and ecosystem they need to thrive independently.

## About the Role

You'll be the voice of developers in the Block ecosystem. As a Developer Advocate, you'll create content, engage communities, and help developers succeed with our APIs. This remote Provo role combines technical depth with community connection—perfect for someone who loves helping others build great things.

## Responsibilities

- Create technical content including blog posts, tutorials, and documentation
- Engage with developers through community events, webinars, and social channels
- Gather developer feedback and communicate needs to product teams
- Build sample applications and code examples showcasing API capabilities
- Support developer onboarding and provide technical guidance

## Requirements

**Experience:** 1-3 years

**Skills:**

- API documentation and sample code
- Community engagement and relationship building
- Public speaking and presentation skills
- Familiarity with payment APIs (preferred)

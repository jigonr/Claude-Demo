## About Cloudflare

Cloudflare is building the foundation of a faster, more secure internet, protecting millions of websites and internet properties daily. Our edge network reaches over 275 cities worldwide, processing over 45 million requests per second.

## About the Role

We're looking for an Infrastructure Engineer I to help design, deploy, and maintain the edge computing infrastructure that powers our global network. You'll work with cutting-edge technology and learn from senior engineers in a supportive environment where growth is expected.

## Responsibilities

- Deploy and configure compute resources across Cloudflare's edge data centers
- Write automation scripts in Python or Go to streamline infrastructure operations
- Monitor system performance and troubleshoot infrastructure issues
- Participate in capacity planning and infrastructure scaling decisions
- Collaborate with security and networking teams on infrastructure hardening
- Document infrastructure architecture and create runbooks for operational procedures

## About the Role

As a Release Engineer at Slack, you'll own the journey that gets code deployed to production safely and quickly. You'll build the automation and processes that enable thousands of engineers to ship with confidence. This hybrid Durham role offers deep technical work combined with cross-team collaboration.

- Monitor deployment metrics, incident response, and system health; implement improvements based on observations
- Build tools and automation to reduce manual release work and improve developer experience
- Document processes and knowledge; mentor engineers on release procedures and best practices
